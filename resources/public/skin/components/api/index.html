接口说明

地图更新接口

地图更新接口用于在游戏里通过脚本自动更新地图信息

地图更新涉及3个元素

  • ID 地图的ID,如果有同ID地图会更新,不能包含空格，推荐用中英文数字
  • Name 地图的名称,通过接口返回的区域名称,不能包含空格，一般时中文
  • Data 地图的数据，多行文本

调用接口时，ID会拼在网址中的:id部分,Name通过mapname的http请求头发送，Data放置在正文部分。

UTF版接口地址: [http://127.0.0.1:4555]/api/utf8update/:id
GBK版接口地址: [http://127.0.0.1:4555]/api/gbkupdate/:id
调用方式：POST

地图搜索接口

地图搜索接口用于在游戏里根据谜题查询具体的区域和房间名

通过多行文字传入谜题正文。

如果查询成功，返回空格分割的区域和房间，比如 苏州 茶馆

如果查询失败，返回空字符串

UTF版接口地址: [http://127.0.0.1:4555]/api/utf8search
GBK版接口地址: [http://127.0.0.1:4555]/api/gbksearch
调用方式：POST
